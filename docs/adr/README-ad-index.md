## Purpose

This index provides a central reference for all A-ZION Architecture Decision Records (AD records). ADs document major architectural decisions, their context, alternatives considered, and consequences.

---

## Scope

This index identifies all known AD records in the A-ZION project. Known AD records currently extend to at least AD-080 based on current project history. Exact AD titles, statuses, and summaries must be extracted from approved source materials.

**Important**: This index is NOT complete. The exact count, full list, and complete content must be confirmed from approved source materials.
